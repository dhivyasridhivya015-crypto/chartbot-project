PROJECT TITLE: Customer Service Chatbot

DESCRIPTION:
This project is a simple chatbot built using Python. It answers basic customer queries like internship, contact, course,fees,location and support.

HOW TO RUN:
1. Open VS Code
2. Run the file using: python chatbot.py
3. Type your question in terminal

FEATURES:
- Simple chatbot
- Keyword based responses
- Internship information
- Course information
- Contact information
- Fee details
- Location details
- Easy to run

TECHNOLOGY USED:
- Python
